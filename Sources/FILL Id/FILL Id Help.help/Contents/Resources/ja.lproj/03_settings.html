FILL Id ヘルプ › 設定

設定

設定はメニューバーのアイコン、または Dock アイコンのメニューから開きます。「設定…」と「ドキュメント表示設定…」の 2 つのウインドウがあります。

設定ウインドウ

メニューの「設定…」で開きます。「一般」と「機能」を 1 枚のウインドウにまとめています。

一般

ログイン時に起動
Mac へのログイン時に FILL Id を自動的に起動します。
表示モード
FILL Id を「メニューバーのみ」「Dock のみ」「メニューバーと Dock」のどこに表示するかを選びます。
ヒント ノッチなどでメニューバーアイコンが隠れてしまったときは、⌘（Command）キーを押しながら FILL Id を起動すると、表示モードを初期状態（メニューバーと Dock）に戻せます。

機能

ホットキー機能の ON/OFF を個別に切り替えます（初期状態はすべてオン）。

  • クリーンペースト（⌥V / ⇧⌥V）
  • 全体表示（⌘0 / ⌥⌘0）
  • ズーム（⌘± / ⇧⌘±）

クリーンペーストのすぐ下には、従属設定として「ペースト時に画面を一瞬暗くする」があります。クリーンペーストの実行を知らせる設定で、クリーンペーストがオフのときは設定できません。

ドキュメント表示設定

メニューの「ドキュメント表示設定…」で開きます。グリッドとガイドの表示切替に関する設定です。

表示切替を有効にする
グリッドとガイドの表示切替機能の ON/OFF。オフのときは切替キーとプリセットの設定が無効になります。
切替キー
表示切替に使うファンクションキーを F1〜F19 から選びます（初期設定は F13）。
設定（プリセット）
切替キーで表示する項目を選びます。ガイド、フレーム枠、文字数、制御文字、フレームグリッド、レイアウトグリッド、ベースライングリッド、ドキュメントグリッドの8項目です。
