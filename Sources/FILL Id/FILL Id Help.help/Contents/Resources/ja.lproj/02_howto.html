FILL Id ヘルプ › 使い方

使い方

FILL Id のホットキーは、InDesign が最前面のときに使えます。各機能は設定の「機能」で個別に ON/OFF できます（初期状態はすべてオン）。

ホットキー一覧

⌥V クリーンペースト
クリップボードの文字を選択範囲へ流し込みます。
⇧⌥V クリーンペースト（前後の空白を除去）
文字の前後の空白や改行を取り除いてから流し込みます。
⌘0 ページ全体表示 ／ ⌥⌘0 見開き全体表示
ウインドウにページや見開きを合わせて表示します。
⌘＋ ／ ⌘− ズームイン／アウト（選択を解除）
選択を解除してから拡大／縮小します。
⇧⌘＋ ／ ⇧⌘− ズームイン／アウト（選択を保持）
選択を保ったまま拡大／縮小します。
F13（初期設定） グリッドとガイドの表示切替
あらかじめ決めた項目の表示／非表示をまとめて切り替えます。切替キーは変更できます。
ヒント InDesign 標準では、全体表示（⌘0 / ⌥⌘0）とズーム（⌘＋ / ⌘−）はメインキーボードの数字キーでしか効きません。FILL Id はこれらをテンキー（数字キーパッド）の 0、＋、− でも実行できるようにします。

クリーンペースト

クリーンペーストは、クリップボードの文字を選択範囲へ流し込む機能です。コピー元のフォントや色などの書式は持ち込まず、貼り付け先（選択した文字の先頭）の書式を引き継いで、文字だけを入れ替えます。

InDesign 標準の「書式なしでペースト」には、ルビが残る、貼り付け先の先頭の書式にならない、互換漢字が統合漢字に変わる、といった挙動があります。クリーンペーストは、その邪魔になりがちな部分を取り除いたものです。InDesign のスクリプト機能で選択範囲の内容を置き換える方式のため、InDesign のペースト挙動に左右されません。

  • 文字は Unicode 正規化（NFC）して流し込みます。結合文字列の合成が目的ですが、互換漢字は変わらないように保護しています。
  • 改行は CR に変換されます。
  • ⇧⌥V では、前後の空白や改行を取り除いてから流し込みます。
メモ 表のセルを選択しているときのペーストには対応していません。

合成できない文字の確認

NFC に正規化しても、合成されずに結合文字や絵文字の修飾子などが残ってしまうことがあります。さらに私用領域の文字など、混入するとトラブルになりそうな文字が含まれていると、確認ダイアログが表示されます。

グリッドとガイドの表示切替

切替キー（初期設定は F13）を押すと、あらかじめ決めたグリッドやガイドの表示／非表示をまとめて切り替えます。

切り替える項目と切替キーは、ドキュメント表示設定で変更できます。
