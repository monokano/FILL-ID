権限とトラブルシューティング

オートメーション（Apple Events）の許可

FILL Id が InDesign を操作するには、オートメーション（Apple Events）の許可が必要です。初めてクリーンペーストなどを実行したときに許可を求めるダイアログが表示されるので、「OK」を選んでください。

許可していない場合は、メニューバーや Dock のメニューに「⚠️ オートメーション権限が必要（InDesign）」と表示されます。クリックすると「システム設定 › プライバシーとセキュリティ › オートメーション」が開きます。FILL Id から InDesign への項目をオンにしてください。

ヒント オートメーションの許可は「FILL Id から InDesign へ」の組み合わせごとに記録されます。InDesign は全バージョンが同じ識別子のため、一度許可すればすべてのバージョンに適用される見込みです。
メモ FILL Id はアクセシビリティ権限を必要としません。

うまく動かないとき

ホットキーがまったく効かない
InDesign が最前面になっているか確認してください。FILL Id のホットキーは、InDesign が最前面のときだけ有効です。
特定の機能だけ効かない
設定の「機能」で、その機能がオンになっているか確認してください。グリッドとガイドの表示切替は、「ドキュメント表示設定」の「表示切替を有効にする」がオンになっている必要があります。
クリーンペーストやズームが動かない
オートメーションの許可（上記）を確認してください。また、全体表示やズームのショートカットは、テンキー（数字キーパッド）の 0、＋、− を使います。
メニューバーアイコンが見当たらない
表示モードが「Dock のみ」になっているか、ノッチでアイコンが隠れている可能性があります。⌘ を押しながら FILL Id を起動すると、表示モードを初期状態（メニューバーと Dock）に戻せます。
