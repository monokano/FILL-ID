FILL Id Help › Settings

Settings

Open Settings from FILL Id's menu bar icon or its Dock icon menu. There are two windows: "Settings…" and "Document Display Settings…".

Settings window

Open it from "Settings…" in the menu. It combines "General" and "Features" in a single window.

General

Launch at Login
Starts FILL Id automatically when you log in to your Mac.
Display Mode
Choose where to show FILL Id: "Menu Bar Only," "Dock Only," or "Menu Bar and Dock."
Tip If the menu bar icon becomes hidden (for example behind the notch), hold the ⌘ (Command) key while starting FILL Id to reset the display mode to the initial state (Menu Bar and Dock).

Features

Turn each hotkey feature on or off individually (all on by default).

  • Clean paste (⌥V / ⇧⌥V)
  • Fit page / spread (⌘0 / ⌥⌘0)
  • Zoom in / out (⌘± / ⇧⌘±)

Just below Clean paste is a dependent setting, "Briefly dim the screen on paste." It signals that clean paste has run, and it is unavailable while Clean paste is off.

Document Display Settings

Open it from "Document Display Settings…" in the menu. These are settings for toggling grids and guides.

Enable display toggle
Turns the grids-and-guides toggle feature on or off. While off, the toggle key and preset settings are disabled.
Toggle key
Choose the function key used for toggling, from F1 to F19 (F13 by default).
Settings
Choose the items shown by the toggle key: Guides, Frame edges, Character count, Invisibles, Frame grids, Layout grids, Baseline grid, and Document grid (eight items).
