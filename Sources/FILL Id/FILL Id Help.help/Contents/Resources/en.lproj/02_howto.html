How to Use

FILL Id's hotkeys work while InDesign is frontmost. You can turn each feature on or off individually under "Features" in Settings (all on by default).

Hotkey list

⌥V Clean paste
Pastes the text on the clipboard into the selection.
⇧⌥V Clean paste (trim surrounding whitespace)
Removes whitespace and line breaks from the start and end of the text before pasting.
⌘0 Fit page / ⌥⌘0 Fit spread
Fits the page or spread in the window.
⌘＋ / ⌘− Zoom in / out (deselect)
Deselects first, then zooms in or out.
⇧⌘＋ / ⇧⌘− Zoom in / out (keep selection)
Zooms in or out while keeping the selection.
F13 (default) Toggle grids & guides
Shows or hides the preset items all at once. The toggle key can be changed.
Tip In InDesign, the shortcuts for Fit (⌘0 / ⌥⌘0) and Zoom (⌘＋ / ⌘−) work only with the main keyboard's number keys. FILL Id also makes them work with the numeric keypad keys 0, +, and −.

Clean paste

Clean paste pastes the text on the clipboard into the selection. It does not bring over the source's formatting such as font or color; instead it keeps the formatting at the paste destination (the first character of the selection) and replaces only the text.

InDesign's built-in "Paste without Formatting" keeps ruby, does not adopt the destination's first-character formatting, and converts compatibility ideographs to unified ideographs. Clean paste removes those often-unwanted behaviors. It replaces the selection's contents with InDesign's scripting, so it does not depend on InDesign's paste behavior.

  • Text is normalized with Unicode NFC before pasting. The purpose is to compose combining character sequences, but compatibility ideographs and similar characters are protected so they are not altered.
  • Line breaks are converted to CR.
  • With ⇧⌥V, whitespace and line breaks at the start and end are removed before pasting.
Note Pasting is not supported when table cells are selected.

Characters that cannot be composed

Even after NFC normalization, combining characters and emoji modifiers can remain without being composed. In addition, when characters that may cause trouble are included, such as Private Use Area characters, a confirmation dialog appears.

Check
Opens a window that finds and shows each such character one by one.
Paste
Performs the clean paste as is.
Cancel
Cancels the clean paste.

Toggle grids & guides

Pressing the toggle key (F13 by default) shows or hides the preset grids and guides all at once.

You can change the items and the toggle key in Document Display Settings.
