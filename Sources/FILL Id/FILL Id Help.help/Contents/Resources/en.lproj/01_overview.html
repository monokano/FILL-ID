About FILL Id

What it does

FILL Id is a utility that performs paste and view operations not built into Adobe InDesign, using hotkeys (keyboard shortcuts). It runs in the background, and its hotkeys are active only while InDesign is frontmost.

It has four main features:

  • Clean paste: Pastes the text on the clipboard into the selection without bringing over any formatting. InDesign's built-in "Paste without Formatting" has unwanted behaviors: it keeps ruby, does not adopt the destination's first-character formatting, and converts compatibility ideographs to unified ideographs. FILL Id's clean paste leaves no ruby, adopts the first-character formatting, and protects compatibility ideographs.
  • Fit page / spread: Fits the page or spread in the window. InDesign's standard ⌘0 works only with the main keyboard's 0, but FILL Id lets you use the 0 on the numeric keypad too.
  • Zoom: Zooms in and out. InDesign's standard ⌘＋ / ⌘− works only with the main keyboard's ＋ / −, but FILL Id lets you use the ＋ / − on the numeric keypad too.
  • Toggle grids & guides: Shows or hides a preset combination of grids and guides all at once.

For how to use each feature, see How to Use. For turning features on or off and other settings, see Settings.

Requirements

FILL Id's hotkeys are active only while InDesign is frontmost. While you work in another app, the hotkeys are inactive and do not interfere with your normal key use.

Note To control InDesign, FILL Id needs Automation (Apple Events) permission. See Permissions and Troubleshooting for details.

Background app and menus

FILL Id stays in the menu bar and the Dock. From its menu bar icon or its Dock icon menu, you can open:

Settings…
Turn features on or off and adjust behavior.
Document Display Settings…
Settings for toggling grids and guides.
Change Log
Shows the change history.

You can choose whether to show FILL Id in the menu bar or the Dock under "Display Mode" in Settings.
